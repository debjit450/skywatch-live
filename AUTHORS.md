## Core Team

## Contributing Communities

We welcome contributions from the community! If you'd like to contribute, please see [CONTRIBUTING.md](CONTRIBUTING.md) for guidelines.

### How to Add Your Name

If you've made a contribution, feel free to:
1. Add your name to this file via a pull request
2. Include your GitHub handle and website (optional)
3. Describe your contributions

### Data Sources

- [OpenSky Network](https://opensky-network.org/) - Flight state data
- [ADS-B One](https://www.adsb.one/) - ADS-B aggregation
- [Airplanes.live](https://airplanes.live/) - ADS-B, military/radar aggregate, UAT, and oceanic ADS-B point sources
- [ADSB.lol](https://adsb.lol/) - ADS-B aggregation
- [ADSBDB](https://www.adsbdb.com/) - Aircraft metadata
- [CelesTrak](https://celestrak.org/) - Satellite TLE data
- [Aviation Weather Center](https://aviationweather.gov/) - Weather and SIGMET
- [Open Glider Network](https://www.glidernet.org/) - Glider tracking
- [FAA](https://www.faa.gov/) - Airspace restrictions
